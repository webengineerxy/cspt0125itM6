## Il Manifesto: Perché un HTML e non un PDF?  

La scelta del formato non è mai neutrale:  

- **PDF**:  
  - Punto fermo  
  - Documento sigillato  
  - "La conversazione finisce qui"  

- **HTML**:  
  - Codice sorgente aperto  
  - Ispezionabile e vivo  
  - Il mio `git commit` alla filosofia **FOSS** (Free and Open Source Software)  

> *"La conoscenza, specialmente quella tecnica, non va imbalsamata, ma condivisa, 'forkata', discussa e migliorata collettivamente."*  

Questo non è solo un esercizio:  
- Invito a guardare "sotto il cofano"  
- Omaggio alla cultura di un web accessibile e democratico  
- Tributo a chi tiene aperte le porte della conoscenza
